# homebrew-dandori

Homebrew tap for [`dandori-cli`](https://github.com/phuc-nt/dandori-cli) — lightweight CLI outer harness for managing AI agent dev teams.

## Install

```bash
brew install phuc-nt/dandori/dandori
```

## Update

```bash
brew upgrade dandori
```

## Maintenance

This repo is auto-updated by `dandori-cli`'s release workflow on every tagged release. Do not edit `Formula/dandori.rb` by hand — changes will be overwritten.
